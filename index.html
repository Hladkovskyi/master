Типы селекторов и их примеры

  • * - универсальный селектор, который выбирает все элементы
  • Селектор по типу элемента - p, span, li и т.д. - выбирает тип элемента
  • class - выбирает элементы на основе значения их атрибута Class
  • id - селектор по идентификатору, который выбирает элементы на основе значения их id атрибута
  • [attr/attr=value и т.д.] - cелекторы по атрибуту предназначены для выбора элементов по имени атрибута и (или) его значению.

      • Псевдоклассы

      • Псевдокласс :link предназначен для выбора не посещённых ссылок.
      • Псевдокласс :visited предназначен для выбора посещённых ссылок.
      • Псевдокласс :active предназначен для выбора элементов в момент когда они активируются пользователем.
      • Псевдокласс :hover предназначен для выбора элементов при поднесении к ним курсора

      По располажению среди соседей

      При выборе элементов можно дополнительно с помощью псевдоклассов задать условие (фильтр) по их расположению (порядковому номеру) в родителе.

      • :first-child - применяется для выбора элементов, которые являются первыми дочерними элементами своего родителя.
      • :last-child - применяется для выбора элементов, которые являются последними дочерними элементами своего родителя.
      • :only-child - используется для выбора элементов, если они являются единственными дочерними элементами внутри родительского контейнера.
      • :only-of-type - применяется для выбора элементов, если каждый из них являются единственным дочерним элементом данного типа внутри своего родителя
      • :enabled - предназначен для выбора включенных элементов формы, с которыми пользователь может взаимодействовать, например, нажать на него или внести текст.
      • :disabled - предназначен для выбора отключенных элементов формы, т.е. элементов с которыми пользователь в данный момент не может взаимодействовать.
